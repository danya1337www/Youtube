# YouTube на UIKit

Учебный проект — интерфейс YouTube на UIKit. Весь UI свёрстан кодом, без сторибордов.

Реализовано:
- лента видео с каруселью шортсов и панелью фильтров по категориям
- полноэкранные шортсы с вертикальным свайпом, лайками, комментариями и подпиской
- переход в полноэкранный режим по тапу на шортс из ленты

Фильтры переключаются, но ленту не фильтруют. Контент (заголовки, просмотры, счётчики) хранится в локальном `Data.json`; превью и аватары подгружаются по сети с picsum, поэтому для отображения картинок нужен интернет с локацией не РФ. Бэкенда нет.
